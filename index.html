Welcome to Alliteration Locator

Alliteration Locator was built as a simple baseline for detecting alliterations. Although it bases it's decisions on orthographics rather than on pronunciation, it is very simple code.

Download source code at https://github.com/bean5/AlliterationLocator /zipball/master.

For compiled versions, you can go to https://github.com/bean5/AlliterationLocator/releases/.

About the code

This is done by using Java.
